# MyBiKE - Modern Bicycle Store Landing Page

This project showcases a stylish landing page designed to help cyclists discover their perfect ride. The site features a fully responsive layout, interactive product sections, and a clean user interface for browsing bikes and accessories.

## Get Started

1. Clone the repository:
```bash
git clone https://github.com/vnechepurenko2005/mybike_landing.git
cd mybike_landing
```
2. Install dependencies:
```bash
npm install
# or
yarn install
```
3. Run the project locally:
```bash
npm start
# or
yarn start
```

## Features ‚ú®

* **Responsive Design**: Optimized layout that adapts seamlessly to mobile phones, tablets, and desktop screens.
* **Scroll Animations**: Elements gracefully fade in and slide up as the user scrolls, powered by the **AOS** library.
* **Navigation**: Fully functional "Burger Menu" for intuitive navigation.
* **Semantic Markup**: HTML5 structure built according to accessibility and SEO best practices.
* **BEM Methodology**: CSS classes named using the Block Element Modifier convention for modular and maintainable code.
* **Interactive UI**: Engaging hover effects on buttons, links, and product cards for better user feedback.
* **Cross-Browser Compatibility**: Verified performance across modern browsers (Chrome, Firefox, Safari, Edge).
